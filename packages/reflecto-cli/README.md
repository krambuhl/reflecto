# reflecto-cli

A cli for developing and statically generating a reflecto application.

## Usage

Develop a reflecto application:

```
reflecto --config .reflecto/config.js --archive .reflecto/archive.js --dev
```

Statically render the reflecto application:

```
reflecto --config .reflecto/config.js --archive .reflecto/archive.js
```

## How to install

With npm, do:

```
npm install reflecto-cli -g
```
