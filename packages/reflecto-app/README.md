# reflecto-app

Contains the reflecto application data store and plugin shell.


## Application layout

__Application__

```
<AppRoot>
    <ShellHeader />
    <PageShell>
        <slot></slot>
    </PageShell>
    <ShellFooter />
</AppRoot>
```

__ComponentRenderer__

```
<Renderer>
    <slot></slot>
</Renderer>
```


## How to install

With npm, do:

```
npm install reflecto-app
```
